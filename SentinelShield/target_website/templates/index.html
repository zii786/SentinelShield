How SentinelShield Works

🔁 Reverse Proxy

All client traffic flows through NGINX, allowing full visibility and control.

🧠 Smart Detection

Detects too many requests, weird headers, or spikes in real time using rules or ML.

🕵️ Tracer

Traces source of threats via IP geolocation and browser fingerprinting—privacy-safe.
